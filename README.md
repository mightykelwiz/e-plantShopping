e-plantShopping

e-plantShopping is a modern web-based plant shopping application built with React.
The application allows users to browse a variety of plants, view plant details and manage a shopping cart in a simple and user-friendly interface.

This project is designed as a frontend-focused application, showcasing clean component structure, reusable UI elements and scalable state management patterns suitable for real-world e-commerce platforms.
